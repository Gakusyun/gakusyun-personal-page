# 背景

在最开始，我使用 GitHub Pages 搭建个人主页，同时这个主页也承载着部分博客的功能。我所有的代码、文字都放在这里，使用 HTML、CSS、JavaScript Native 构建静态页面。

后来接触到 Vercel，并使用 [Fuwari](https://github.com/saicaca/fuwari) 搭建了这个博客。

但是，我还不满足，又用 Vue 搭建了个人主页，但是由于个人主页存储在另一个仓库，导致年久失修，时常更新不及时。

# 合并

但是现在不会了，在 GLM 4.7 的帮助下，我将我的博客和个人主页合二为一，使体验不再割裂。

在工作过程中，AI 起到了举足轻重的作用，但是仍然有部分内容需要手动修改。

# 总结

总体来说，合并已经完成。同时，我宣布 blog.gxj62.cn 会重定向到 www.gxj62.cn，将来会完全关闭 blog.gxj62.cn。
